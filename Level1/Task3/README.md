# 🌡️ Temperature Converter

A responsive web application that converts temperatures between Celsius, Fahrenheit, and Kelvin.

## 🚀 Features

- Celsius ↔ Fahrenheit
- Celsius ↔ Kelvin
- Fahrenheit ↔ Kelvin
- Input Validation
- Responsive UI
- Clean and Simple Design

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
